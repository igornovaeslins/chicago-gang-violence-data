# Data dictionary

All files are CSV unless noted. The spatial unit is the Chicago **community
area** (77 areas, codes 1–77); the temporal unit is the calendar **year**
(2008–2024). 2015 is dropped from the analyses (see README, Construction notes).

---

## `data/processed/chicago_painel_completo.csv`

One row per community area × year (1,309 rows). Built by
`scripts/47_painel_chicago_consolidado.R` by joining the base panel, the ACS
controls, and the shooting/DV decompositions.

| Column | Description |
|---|---|
| `community_area`, `community_area_name` | Community-area code (1–77) and name |
| `year` | Calendar year |
| `gang_dominant` | Modal dominant gang in the area (CPD CLEARMAP maps) |
| `gang_family` | Gang family of `gang_dominant` (Folk Nation / People Nation / other) |
| `n_gangs_present` | Count of distinct gangs mapped in the area that year |
| `pct_area_gang_control` | Share of area land covered by mapped gang territory (0–1) |
| `gang_map_year_used`, `imputed_gang_map` | Map year used; flag for the 2011/2013 nearest-year fallback |
| `n_dv_crimes` | Domestic-violence incidents recorded |
| `n_dv_res`, `n_dv_pub` | DV incidents in residences / in public space |
| `n_dv_homicide` | DV-flagged homicides |
| `n_dv_arrested` | DV incidents recorded as ending in arrest |
| `arrest_rate_dv` | `n_dv_arrested / n_dv_crimes` |
| `n_dv_check`, `dv_consistency_ratio` | Internal cross-checks from the build |
| `n_homicides` | All homicides in the area-year |
| `n_arrests` | Total arrests (CPD dataset `dpt3-jri9`, begins 2014) |
| `n_shoot_total` | All shooting victims |
| `n_fem_shoot`, `n_male_shoot` | Shooting victims by sex |
| `n_fem_shoot_res`, `n_fem_shoot_pub` | Female shooting victims in residences / public space |
| `n_fem_shoot_fatal` | Fatal female shooting victims |
| `total_pop`, `pop_black`, `pop_white`, `pop_latino`, `pop_female` | ACS population counts |
| `pct_black`, `pct_white`, `pct_latino`, `pct_female` | ACS population shares |
| `median_hh_income`, `per_capita_income` | ACS income |
| `pct_poverty`, `pct_unemp_young`, `pct_bachelor_plus` | ACS poverty, young-adult unemployment, bachelor's-or-more share |
| `area_km2`, `pop_density_per_km2` | Area and density |
| `dv_per_100k`, `fem_shoot_per_100k_female` | Per-capita rates |

## `data/processed/chicago_acs_ca_year.csv`

ACS 5-year demographic estimates interpolated to community area × year, from
Census tract-level extracts (vintages 2010, 2014, 2018, 2022) mapped to
community areas. Merged into the panel by script 47.

## How the intermediate files were built

The intermediate files were produced from the public portals by a download and
spatial-join step: CPD crimes and shooting victims via the Socrata API, CLEARMAP
gang polygons, ACS via the Census API, then a point-in-polygon join of each
incident to its community area and to the dominant gang. Those steps require
internet access and are not part of the offline path here, which starts from the
intermediate files. The download/join code can be shared on request.
